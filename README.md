# Netflix Data Analysis

## Project Overview

This project explores the Netflix Movies and TV Shows dataset to uncover insights about content distribution, production trends, and genre popularity on Netflix.

Using Python data analysis tools, the dataset was cleaned, explored, and visualized to understand how the streaming catalog has evolved over time.

---

## Objectives

* Analyze the distribution of Movies vs TV Shows
* Identify the countries producing the most Netflix content
* Examine how Netflix content has grown over the years
* Discover the most common genres on the platform
* Identify top directors contributing to Netflix titles

---

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook (Anaconda)

---

## Dataset

The dataset contains information about Netflix titles including:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Release Year
* Genre
* Duration

---

## Key Analysis Performed

1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Visualization of content distribution
4. Trend analysis of Netflix content growth

---

## Key Insights

* Movies make up the majority of Netflix content compared to TV Shows.
* Content production on Netflix increased significantly after 2015.
* The United States produces the highest number of titles on Netflix.
* Drama and International Movies are among the most common genres.
