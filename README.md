# Netflix Data Analysis

This project explores Netflix's movie and TV show dataset to identify trends in streaming content.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Questions
- What type of content dominates Netflix (Movies vs TV Shows)?
- Which countries produce the most Netflix content?
- What are the most common genres?
- How has Netflix content grown over the years?

## Dataset
Netflix Movies and TV Shows dataset containing information about titles, release year, rating, country, genre, and duration.
